# another CHIP-8 emulator

![Demo - Octojam 1 title ROM](screenshots/octojam1-title.png)

## how to use

```
git clone https://github.com/s-m33r/anotherchip8.git
```

```
pip install -r requirements.txt
python main.py <path to ROM>
```
